CÓMO CAMBIAR LAS IMÁGENES

1. Abre la carpeta: imagenes
2. Reemplaza estos archivos por tus imágenes:
   - foto-gemelas.svg  -> puedes usar foto-gemelas.jpg o foto-gemelas.png
   - rosas-rojas.svg   -> puedes usar rosas-rojas.png
   - vestido-quince.svg -> puedes usar vestido-quince.png
   - mariposa.svg      -> puedes usar mariposa.png

3. Si cambias la extensión, también cambia el nombre en index.html.
   Ejemplo:
   Antes: <img src="imagenes/foto-gemelas.svg">
   Después: <img src="imagenes/foto-gemelas.jpg">

RECOMENDACIÓN
- Para el vestido usa PNG con fondo transparente.
- Para la foto de las gemelas usa JPG o PNG cuadrada.
- Para las rosas usa PNG con fondo transparente.
